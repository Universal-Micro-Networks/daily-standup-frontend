# Welcome to Daily Standup Frontend

This is the storybook documentation for the Daily Standup Frontend application.

## Overview

This application is built with React, TypeScript, and Tailwind CSS. It provides a platform for daily standup meetings and team collaboration.

## Features

- User authentication
- Daily report creation and management
- Team management
- Multi-language support
- Theme customization
- Responsive design

## Getting Started

To get started with this application, please refer to the individual component stories for detailed usage examples.
